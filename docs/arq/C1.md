# C1 - Contexto do Sistema

## Visao Geral
O sistema "Analise Financeira" permite consultar cotacoes e historico de ativos da B3, mostrando dados em uma interface web simples.

## Atores
- Usuario: consulta ativos e historicos.

## Sistemas Externos
- brapi.dev API: fornece cotacoes e historico.

## Principais Interacoes
- Usuario acessa a aplicacao web no navegador.
- Frontend chama a API interna do backend.
- Backend consulta a brapi.dev para dados financeiros.
