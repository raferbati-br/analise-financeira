# C4 - Detalhamento (Codigo)

## Estrutura de Arquivos
- server.js
  - Servidor HTTP, integracao com SDK da brapi.dev, rotas REST.
- public/index.html
  - Layout da pagina e estrutura das tabelas.
- public/app.js
  - Logica de UI, chamadas API e renderizacao.
- public/styles.css
  - Estilos e layout.

## Endpoints
- GET /api/tickers
  - Retorna lista de tickers (filtrada por tipo).
- GET /api/quote?symbols=...
  - Retorna cotacoes para ate 10 tickers.
- GET /api/history?symbol=...&range=...&interval=...
  - Retorna historico de preco para um ticker.
