# C4 - Detalhamento (Codigo)

## Estrutura de Arquivos
- server.js
  - Servidor HTTP, integracao com SDK da brapi.dev, rotas REST.
- public/index.html
  - Layout da pagina e estrutura das tabelas.
- public/app.js
  - Logica de UI, cache local, chamadas API e renderizacao (orquestra os modulos).
- public/analysis/utils.js
  - Funcoes utilitarias para series de preco.
- public/analysis/trend.js
  - Calculo de tendencia.
- public/analysis/patterns.js
  - Identificacao de padroes graficos.
- public/analysis/volume.js
  - Confirmacoes por volume e nivel de preco.
- public/analysis/momentum.js
  - Calculo de momento do preco.
- public/analysis/risk.js
  - Calculo de risco/retorno.
- public/analysis/score.js
  - Score por confluencia de fatores.
- public/styles.css
  - Estilos e layout.

## Endpoints
- GET /api/tickers
  - Retorna lista de tickers (filtrada por tipo).
- GET /api/quote?symbols=...
  - Retorna cotacoes para ate 10 tickers.
- GET /api/history?symbol=...&range=...&interval=...
  - Retorna historico de preco para um ticker.
