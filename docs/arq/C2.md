# C2 - Containers

## Containers
- Frontend (HTML/CSS/JS estatico)
  - Responsavel por UI, selecao de tickers, exibicao de cotacoes e historico.
- Backend (Node.js HTTP server)
  - Exp?e endpoints REST e integra com SDK da brapi.dev.
- API Externa (brapi.dev)
  - Fonte de dados de mercado.

## Fluxo Principal
1) Usuario abre a pagina do frontend.
2) Frontend requisita lista de tickers em /api/tickers.
3) Usuario seleciona ativos e solicita cotacoes.
4) Frontend chama /api/quote e /api/history.
5) Backend busca dados na brapi.dev via SDK e responde ao frontend.
