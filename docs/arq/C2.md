# C2 - Containers

## Containers
- Frontend (HTML/CSS/JS estatico) - `frontend/`
  - Responsavel por UI, resumo, tabela paginada, filtros, ordenacao e logos.
- Modulos de Analise (JS ES Modules) - `frontend/analysis/`
  - Calculos de tendencia, padroes, volume, momento, risco e score.
- Backend (Node.js HTTP server) - `backend/`
  - Exp?e endpoints REST e integra com SDK da brapi.dev.
- API Externa (brapi.dev)
  - Fonte de dados de mercado.

## Fluxo Principal
1) Usuario abre a pagina do frontend.
2) Frontend requisita lista de tickers em /api/tickers.
3) Frontend busca historico (3mo) por ticker via /api/history.
4) Frontend calcula analises e monta o resumo e a tabela (com logos e ordenacao).
5) Backend busca dados na brapi.dev via SDK e responde ao frontend.
