# C3 - Componentes (Backend)

## Componentes
- HTTP Server (`backend/server.js`)
  - Roteamento das rotas /api/quote, /api/tickers, /api/history.
- Brapi Client
  - Cliente do SDK para chamadas de cotacoes e lista de ativos (`backend/api/brapi.js`).
- Cache de Tickers
  - Cache simples em memoria para reduzir chamadas de lista.
- Modulos de Analise (frontend)
  - Funcoes separadas para tendencia, padroes, volume, momento, risco e score.
- UI de Resumo e Tabela (frontend)
  - Abas, cards de resumo com logos, tabela com ordenacao e painel de score por item.
- Cache Local (frontend)
  - localStorage com TTL para evitar novas chamadas ao recarregar.
- Env Loader (`backend/env.js`)
  - Le e aplica variaveis do arquivo .env.

## Responsabilidades
- Validar parametros de entrada.
- Normalizar tickers (ex.: remove sufixos).
- Converter erros do SDK para mensagens amigaveis.
